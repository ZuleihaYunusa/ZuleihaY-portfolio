## How to Deploy to GitHub

To get this portfolio "live" on GitHub, follow these steps:

### 1. Create a New Repository on GitHub
1. Log in to your GitHub account: [github.com](https://github.com/)
2. Click the **+** icon in the top right and select **New repository**.
3. Name it exactly `zuleihayunusa.github.io` (or just `my-portfolio`).
4. Keep it **Public**.
5. Do NOT initialize with a README, .gitignore, or license (we already have them).
6. Click **Create repository**.

### 2. Push Your Code
Open your terminal (PowerShell or Command Prompt) in this folder and run:

```bash
# Initialize git
git init

# Add all files
git add .

# Commit changes
git commit -m "Initial portfolio setup"

# Link to your GitHub repo (Replace <URL> with your repo URL)
git remote add origin https://github.com/zuleihayunusa/zuleihayunusa.github.io.git

# Push to GitHub
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to your repository on GitHub.
2. Click **Settings** > **Pages**.
3. Under **Branch**, select `main` and `/ (root)`.
4. Click **Save**.
5. Your site will be live at `https://zuleihayunusa.github.io/` within a few minutes!
